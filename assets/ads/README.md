# Assets publicitaires locaux

Placez ici uniquement les visuels publicitaires locaux du portail.

Regles:

- Formats: `.webp`, `.jpg`, `.jpeg`, `.png`.
- Pas de SVG annonceur externe.
- Pas de scripts, pixels, iframes ou tags de tracking.
- Poids conseille: moins de 250 Ko par image.
- Nom de fichier simple: lettres, chiffres, tirets et underscores.

Les images peuvent ensuite etre referencees dans `ads.json` avec un chemin du type:

```json
"image": "assets/ads/offre-boutique.webp"
```
